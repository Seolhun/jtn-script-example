# Localize Components

## Used stacks

- TypeScript
- Babel 7
- Lerna
- RollupJS
- React
- EmotionJS
- Storybook

## How to getting started
#### Install Package

```bash
$ npm install @seolhun/localize-components
```

## How to use
#### import
```js
import { LocalizeModal } from '@seolhun/localize-components';
```
