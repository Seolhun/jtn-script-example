# Localize Components Grid

## Used stacks

- TypeScript
- Babel 7
- Lerna
- RollupJS
- React
- EmotionJS
- Storybook

## How to getting started
#### Install Package

```bash
$ npm install @seolhun/localize-components-grid
```

## How to use
#### import
```js
import { LocalizeRow, LocalizeCol } from '@seolhun/localize-components-grid';
```
