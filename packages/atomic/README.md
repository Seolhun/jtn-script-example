# Localize Components

## Used stacks
- React
- TypeScript
- EmotionJS
- Docz

## How to getting started
#### Install Package
```bash
$ npm install @seolhun/localize-components
```

#### Run Docs
```bash
$ npm run dev
```

## How to use
#### import
```js
import { Button, anything } from '@seolhun/localize-components';
or 
import * as Components from '@seolhun/localize-components';
```
