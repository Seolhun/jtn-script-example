# Localize Components Atomic

## Used stacks

- TypeScript
- Babel 7
- Lerna
- RollupJS
- React
- EmotionJS
- Storybook

## How to getting started
#### Install Package

```bash
$ npm install @seolhun/localize-components-atomic
```

## How to use
#### import
```js
import { LocalizeButton } from '@seolhun/localize-components-atomic';
```
