# Localize Components

## Used stacks
- React
- TypeScript
- SCSS
- Storybook

## How to getting started
#### Install Package
```bash
$ yarn add @seolhun/localize-components
```

#### Run Docs
```bash
$ yarn run dev
```

## How to use
#### import
```js
import { Button, anything } from '@seolhun/localize-components';
or
import * as Components from '@seolhun/localize-components';
```
